Better Ponymotes

About

Better Ponymotes is a browser addon to display ponymotes across Reddit. This means things like [](/c01) will work outside of r/mylittlepony, but also that ponymotes from other subreddits (such as [](/rose6)) will work across the site.

Installation

Firefox Installation

Install the addon from betterponymotes.xpi. Firefox should handle the file automatically; if not, drag the file onto Firefox and it will install then.

Chrome Installation

Install the addon from betterponymotes.crx.

Be warned that beta versions of Chrome are changing the way extension installation works- it will no longer be possible to automatically install addons from sites without hosting them through the Chrome Web Store. As I may not be doing this, you will have to install it manually, by dragging the file onto the extensions page at about:extensions.

Updates

No matter how you install BetterPonymotes, it will be automatically updated just like any other addon. Your browser will periodically check this site for new versions, and download them if available. The addon does not normally require a restart for updates, however, you may have to refresh pages before most new features will work (including new ponymotes).

Configuration

BetterPonymotes has two options you can change, controlling whether or not certain ponymotes/features are enabled.

In Firefox, these options can be found by clicking Preference in the Addons Manager (about:addons). In Chrome, click Options from the Extensions (about:extensions) page.

The Enable NSFW Emotes option does as it says- enables or disables display of a handful of questionable ponymotes. These aren't used very frequently across Reddit (especially the pony subreddits, which tend toward being more SFW), but be warned that some of them are very explicit.

The Enable Extra CSS option enables the "chaos script", a collection of emote flags and modifiers such as -zspin, which apply various CSS transformations. You may want to disable this if your computer is less capable of handling them, as they're known to bog down browsers.

These options normally take effect immediately in Firefox without reloading any pages. Chrome requires a refresh.
